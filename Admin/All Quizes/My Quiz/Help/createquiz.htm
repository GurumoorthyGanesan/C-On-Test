Creating a quiz

To create a quiz yourself, follow these steps:

1. Start the Quiz Administrator 

2. Click on File > New Quiz

3. Type the following questions In the question textbox -

If 4 DRAX=5 LINX and 2 LINX=6 PINX. 24 DRAX will equal how many PINX ?
90
60
45
80

2/3

1
0
2

4. Now go to the user option. Here click on the Add User button and enter a username and password.

5. To set the options click on the option icon. Set the title to Sample Quiz. And type the time : 5.

6. Now save the quiz by clicking File > Save Quiz

7. Quit Quiz Administrator and Start Quiz. Choose Sample Quiz from the quiz list. Now enter your password in the password field. Click on OK to start.
