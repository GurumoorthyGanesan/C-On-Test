Interface Overview

When you first start the quiz administrator program you will see the following screen. Here you have to enter the question set. The image shows how should you organize your question set.

When you click on the User item on the tasks bar, you will see the following screen. Here you have to decide who is going to take the test. You can add or remove a contestant.

The Options task lets you set some of the quiz's option. Here you can set the title of your quiz, set the passing level, decide about quiz type and set the time.

The Results option lets you know the results of the test. When a person attends any quiz, his scores are kept. It is later shown here.

Sometimes you may need to change quiz name or delete a quiz. In that time the Manage Quiz option will let you do these.

When you are attending the quiz, you are first presented with the following login dialog box. You have to choose the quiz you want to appear, choose your username and enter the appropriate password.

If you enter the appropriate password, you will see the quiz screen like the following image. Here all the necessary information and the question and answers will be showed.

When you have done with the quiz, a result about your quiz will be shown like the image below. Here you can see the result.
